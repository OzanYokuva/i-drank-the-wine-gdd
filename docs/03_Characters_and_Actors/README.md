## ğŸ“ Character Documentation Standards

Each character profile should include:

### Required Sections
1. **Quick Reference** - Key stats and identity at a glance
2. **Character Identity** - Core concept, duality, physical presence
3. **Background & History** - Early life, pre-game life, breaking point
4. **Combat Profile** - Fighting philosophy, skill sets, statistics
5. **Personality & Psychology** - Core traits, voice, relationships
6. **Character Arc** - Story progression across acts
7. **Visual Design** - Appearance, animation notes
8. **Gameplay Role** - Player fantasy, playstyle, difficulty curve
9. **Narrative Connections** - Themes, other characters, philosophy
10. **Development Notes** - Status, future plans, voice acting

### Recommended Sections
- Philosophy integration
- Key character moments
- Future story potential
- Character checklist
- Quick gameplay tips

## ğŸ­ Shared Themes

All playable characters share these core elements:

### Identity & Inheritance
- Born into modern world, unaware of bloodline
- Society buried their true identities under false names
- Rage/fury as fire burning away false layers
- Discovery of ancestral warrior heritage

### The Cycle
- Each fights independently, unaware of others
- All face similar triggers (corruption, loss, injustice)
- All must learn violence perpetuates the cycle
- Understanding replaces fury as true weapon

### Philosophical Journey
1. **Descent** (Eckhart/Jung) - Confront darkness
2. **Cycle** (Buddha) - Recognize pattern
3. **Identity** (Game Motto) - True self emerges
4. **Balance** (Resolution) - Become balancer, not destroyer

## ğŸ® Combat Design Philosophy

### Three Skill Sets Per Hero
Every playable character features three distinct combat styles:

1. **Barehand/Core Style** - Character's natural fighting method
2. **Close-Range Weapon** - Specialized weapon type
3. **Heavy/Extended Weapon** - Alternative playstyle

This triples variety and replayability per character.

### Design Goals
- **Accessibility:** Easy to learn, rewarding to master
- **Distinctiveness:** Each character feels unique
- **Depth:** Multiple viable builds and strategies
- **Narrative Integration:** Combat reflects personality

## ğŸ’¡ Design Principles

### Character Creation Guidelines

**For Playable Heroes:**
1. Start with clear core concept
2. Define duality (before/now/true self)
3. Establish distinct combat identity
4. Integrate philosophical themes
5. Connect to global narrative

**For NPCs:**
1. Clear role in player's journey
2. Memorable personality trait
3. Visual distinctiveness
4. Dialogue voice consistency

**For Enemies/Bosses:**
1. Embody stage/area theme
2. Fair but challenging mechanics
3. Clear visual telegraphs
4. Narrative significance
