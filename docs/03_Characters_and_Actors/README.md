# 03 - Characters and Actors

## Overview

Complete documentation for all characters, NPCs, enemies, and interactive actors.

## Sections

- Playable Characters - Player-controlled characters
- NPCs - Non-player characters
- Enemies and Bosses - Antagonists and challenges

## Character Documentation Standards

Each character entry should include:
- Name and title
- Role in the game
- Background and lore
- Abilities and stats
- Relationships with other characters
